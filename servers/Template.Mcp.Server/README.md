# Microsoft Template MCP Server

A template project for creating new MCP servers. This template provides the foundational structure for building custom Model Context Protocol servers.

## Table of Contents
- [Overview](#overview)
- [Usage](#usage)
- [Contributing](#contributing)

# Overview

This is a template project. It demonstrates the core structure needed to build an MCP server but does not include any tools by default.

# Usage

Use this template as a starting point for building your own MCP server with custom tools.
