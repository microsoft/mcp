# Sovereign Cloud Support Status

This document tracks the implementation status of sovereign cloud support across all Azure MCP service areas. Sovereign clouds (Azure Government, Azure China, etc.) require special endpoint configurations and authentication handling. Each service area must be updated to properly support these environments.
